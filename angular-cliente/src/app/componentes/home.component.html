CMS

CMS, manejador de contenidos armado con Express, Mongo y Angular

Caracteristicas:

  • Sistema de login con autenticacion aplicando JWT.
  • Sistema de gestion interna de data con comunicación directa con la API.
  • Dashboard de visualización de los datos ingresados.
  • Extensibilidad

Soon:

  • Integración con aplicación de escritorio a traves de Electron.
  • Visualización de datos con libreria D3.js.
